Security Features

Built for Academic Integrity

Every exam runs in a completely isolated container with enterprise-grade security controls.

Secure Containers

Each student gets their own isolated Linux desktop environment. No cross-contamination, no shared resources, complete process isolation.

Clipboard Blocking

RFB protocol-level filtering blocks all clipboard transfers between the exam environment and the outside world. No copy-paste cheating possible.

Website Filtering

DNS-based allowlisting ensures students can only access approved resources. Block the entire internet or allow specific documentation sites.
